What is the PGN Translator

The PGN Translator helps bridge the language barrier in the chess community. This project is designed to provide chess players worldwide with a tool to translate chess game notations, known as PGN (Portable Game Notation).

Support Multiples Languages

It has the ability to translate into multiple languages while maintaining the original format for an easy process of chess game data using PGN.

Here are the languages supported:

English, Czech, Danish, Dutch, Estonian, Finnish, French, German, Hungarian, Icelandic, Italian, Norwegian, Polish, Portuguese, Romanian, Spanish and Swedish.
